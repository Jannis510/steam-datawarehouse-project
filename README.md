# steam-datawarehouse-project
Dieses Repository enthält die Abgabe für das Modul Data Warehouse. Im Projekt wird ein Data Warehouse auf Basis der Steam News API und SteamSpy aufgebaut und mittels ETL-Prozessen befüllt. Ziel ist die Analyse von spielbezogenen Kennzahlen und deren zeitlichem Zusammenhang mit veröffentlichten News (u. a. Popularität und Aktivität).
